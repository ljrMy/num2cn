# 数字转中文 Number to Chinese

1.支持正数
```python
chinese(12345)
#一万二千三百四十五
```

2.支持0的读法
```python
chinese(0)
#零
chinese(1004)
#一千零四
chinese(10208)
#一万零二百零八
chinese(1200)
#一千二百
```

3.支持大数
```
chinese(386492865426526854932864846584992)
#三沟八千六百四十九穰二千八百六十五秭四千二百六十五垓二千六百八十五京四千九百三十二兆八千六百四十八亿四千六百五十八万四千九百九十二
```

4.支持负数
```
chinese(-998)
#负九百九十八
```

5.支持小数
```
chinese(1.667)
#一点六六七
```
